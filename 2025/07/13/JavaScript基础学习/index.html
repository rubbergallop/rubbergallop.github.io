基本语法

var：函数作用域（伪全局作用域），声明的变量会变量提升（预解析，即会提前加载变量，但不会加载赋值过程）

let：块级作用域，不能重复声明

const：常量

输出：undefined即变量还未赋值，null即变量赋初值为null

条件语句：if,else,else if

循环语句：for,while,do while

循环关键字：break,continue

事件

事件指窗口中发生的特定瞬间，例如点击鼠标，按下键盘，加载页面等

事件 描述
onClick 点击事件
onMouseOver 鼠标经过
onMouseOut 鼠标移出
onChange 文本内容改变事件
onSelect 文本框选中
onFocus 光标聚集
onBlur 移开光标
